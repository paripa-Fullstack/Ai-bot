# 🤖 AI Premium Telegram Bot

## 🚀 Описание

Премиум Telegram-бот с интеграцией мощного AI (DeepInfra) и системой подписок на криптовалюту!

### ✨ Возможности бота:

- 🤖 **Мощный AI-ассистент** на базе Meta-Llama-3.1-70B
- 💎 **Система подписок** с оплатой в криптовалюте (TRC20, BTC)
- 🎁 **5 бесплатных запросов** для новых пользователей
- 📊 **Полная статистика** использования
- 💾 **База данных** SQLite с хранением всех данных пользователей
- 📱 **Сбор контактов**: телефон, геолокация, имя
- 📜 **История запросов** пользователя
- 💳 **Ручная проверка платежей** администратором

### 💡 Что умеет AI:

- Отвечает на любые вопросы
- Помогает с программированием
- Создает тексты, статьи, контент
- Переводит на любые языки
- Решает математические задачи
- Дает советы и рекомендации
- Генерирует идеи для бизнеса

## 📦 Установка

### 1. Установите Python 3.8+

Убедитесь, что Python установлен:
```bash
python3 --version
```

### 2. Создайте виртуальное окружение (рекомендуется)

```bash
python3 -m venv venv
source venv/bin/activate  # Для Linux/Mac
# или
venv\Scripts\activate  # Для Windows
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Запустите бота

```bash
python3 ai_telegram_bot.py
```

## 💎 Тарифные планы

- 🎁 **Пробный период**: 5 бесплатных запросов
- 💵 **Неделя**: $5 (безлимит на 7 дней)
- 💵 **Месяц**: $15 (безлимит на 30 дней)
- 💵 **Год**: $100 (безлимит на 365 дней)

## 📊 База данных

Бот создает файл `bot_users.db` с тремя таблицами:

### 1. users (Пользователи)
- user_id - ID пользователя
- username - Имя пользователя в Telegram
- first_name, last_name - Имя и фамилия
- phone - Номер телефона
- location_lat, location_lon - Координаты
- name - Имя, указанное при регистрации
- registration_date - Дата регистрации
- trial_uses - Оставшиеся бесплатные использования
- is_subscribed - Статус подписки
- subscription_end - Дата окончания подписки
- total_requests - Всего запросов

### 2. payments (Платежи)
- id - ID платежа
- user_id - ID пользователя
- subscription_type - Тип подписки
- payment_date - Дата платежа
- receipt_file_id - ID файла чека
- approved - Статус одобрения

### 3. request_history (История запросов)
- id - ID записи
- user_id - ID пользователя
- request_text - Текст запроса
- response_text - Ответ AI
- request_date - Дата запроса

## 🎯 Команды бота

- `/start` - Начать работу (регистрация)
- `/help` - Помощь и информация
- `/status` - Проверить свой статус
- `/subscribe` - Оформить подписку
- `/history` - Последние 10 запросов

## 💳 Процесс оплаты

1. Пользователь выбирает `/subscribe`
2. Выбирает тарифный план
3. Получает крипто-адреса для оплаты
4. Отправляет скриншот транзакции
5. Администратор проверяет и активирует подписку

## 🔐 Безопасность

- Все данные хранятся локально в SQLite
- Номера телефонов и геолокация защищены
- Платежи требуют ручной проверки администратором

## 🛠️ Администрирование

### Активация подписки пользователю вручную

Подключитесь к базе данных:
```bash
sqlite3 bot_users.db
```

Активировать подписку:
```sql
-- Для недельной подписки
UPDATE users 
SET is_subscribed = 1, 
    subscription_end = datetime('now', '+7 days') 
WHERE user_id = USER_ID;

-- Для месячной подписки
UPDATE users 
SET is_subscribed = 1, 
    subscription_end = datetime('now', '+30 days') 
WHERE user_id = USER_ID;

-- Для годовой подписки
UPDATE users 
SET is_subscribed = 1, 
    subscription_end = datetime('now', '+365 days') 
WHERE user_id = USER_ID;
```

Одобрить платеж:
```sql
UPDATE payments 
SET approved = 1 
WHERE id = PAYMENT_ID;
```

### Просмотр всех пользователей:
```sql
SELECT user_id, username, phone, trial_uses, is_subscribed, subscription_end 
FROM users;
```

### Просмотр всех платежей:
```sql
SELECT * FROM payments ORDER BY payment_date DESC;
```

### Статистика:
```sql
-- Всего пользователей
SELECT COUNT(*) FROM users;

-- Активные подписки
SELECT COUNT(*) FROM users 
WHERE is_subscribed = 1 AND subscription_end > datetime('now');

-- Всего запросов
SELECT SUM(total_requests) FROM users;

-- Неодобренные платежи
SELECT * FROM payments WHERE approved = 0;
```
